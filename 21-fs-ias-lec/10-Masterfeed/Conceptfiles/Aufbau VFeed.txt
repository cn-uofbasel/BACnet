
Classical Conzept:

+-event------------------------------------------------------------------+
| +-meta---------------------------------------+                         |
| | feed_id, seq_no, h_prev, sign_info, h_cont |, signature, opt_content |
| +--------------------------------------------+                         |
+------------------------------------------------------------------------+

Virtual Feed Concept:

+-event--------------------------------------------------------------------+
| +-meta---------------------------------------+                           |
| | feed_id, seq_no, h_prev, sign_info, h_cont |, signature, virtual_event |
| +--------------------------------------------+                           |
+--------------------------------------------------------------------------+

the virtual_event is written inside the opt_content, it is build the same like the event.
it's an event inside the event:

+-virtual_event-----------------------------------------------------------------+
| +-meta--------------------------------------------+                           |
| | vfeed_id, vseq_no, vh_prev, vsign_info, vh_cont |, vsignature, vopt_content |
| +-------------------------------------------------+                           |
+-------------------------------------------------------------------------------+
